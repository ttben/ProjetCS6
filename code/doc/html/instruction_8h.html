enum Condition

Conditions.

Ces valeurs sont associées à l'instruction de branchement (BRANCH et CALL) et déterminent la condition à tester par référence à la valeur du code condition (Condition_Code).

Une valeur de ce type remplace le numéro de registres dans les isntructions BRANCH et CALL.

Valeurs énumérées
NC 

Pas de condition (nrachement inconditionnel)

EQ 

Égal à 0.

NE 

Différent de 0.

GT 

Strictement positif.

GE 

Positif ou nul.

LT 

Strictement négatif.

LE 

Négatif ou null.
